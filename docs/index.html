Audio File Formats

Here are the most common audio file formats, their approximate file sizes, and their categories.
Format Type Subcategory Approx. Size (1 min, stereo) Notes
MP3 Lossy CBR, VBR 0.7–1.5 MB (128–320 kbps) Most popular for music download/streaming
AAC Lossy CBR, VBR 0.7–1.2 MB (128–256 kbps) Default for iTunes, YouTube, Apple Music
OGG Vorbis Lossy CBR, VBR ~1 MB (160 kbps) Open, free, used in games/Spotify
WAV Lossless Uncompressed PCM ~10.1 MB (16-bit/44.1 kHz) Studio/archival, very large files
FLAC Lossless Compressed ~5–7 MB Popular for high-quality music
ALAC Lossless Compressed ~5–7 MB Apple’s lossless, for iTunes/Apple devices
Opus Lossy VBR ~0.5–1 MB (48–128 kbps) Modern, low-latency (VoIP, Discord, streaming)
AIFF Lossless Uncompressed PCM ~10.1 MB (16-bit/44.1 kHz) Apple’s uncompressed, pro audio
Tip: Lossy formats are much smaller, but lose some quality. Lossless preserve all original data, but are much larger.
